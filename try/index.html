Keybindings of the console:
      Enter          send the current line to the interpreter
      Up / Ctrl-P    move back in history
      Down / Ctrl-N  move forward in history
      Ctrl-L         clear transcript
      Ctrl-D         end-of-file when line is empty
      Ctrl-C         interrupt execution
